Design Tokens Pro is a digital course aimed at designers, developers, and stakeholders involved in a design system and product design.

It provides a reliable guide for establishing the best management workflow for your specific context.

If the management of design tokens across different files or tools appears challenging, this course is for you.

What's inside the course

What you'll learn

Develop a shared vocabulary. Creating a shared taxonomy with clear naming conventions is vital. It organizes information, enhances clarity, and promotes teamwork and efficiency.

Translate values into design decisions. Every token you make is a careful choice, not just a random number. As you learn more, you'll see how and when you make these important choices and use this understanding in real situations.

Select the right token for each case. Choosing the right token is vital for different use cases, as it ensures functionality. Select based on each use case's specific needs and be aware of any potential implications.

Optimize your process. Understanding the tools for efficient design token updates is key. As products evolve, so must these tokens. Developing skills to manage and update these tokens can greatly enhance the design process.

Questions + Answers

Is it suitable for beginners?

Yes, the guide is written in a clear and easy-to-understand manner for those starting from scratch. Even experienced designers will find new insights and tips in the guide.

What problems will this course help me overcome?

This course makes it easier for you to use design tokens. It helps keep your digital projects consistent, understand how to use design tokens, and simplify your design process. It also offers tips for managing design tokens across different tools or files.

Can I do the lessons when I have time?

Yes, this is a digital and self-paced course, you can take as long as you want to complete it.

Should I know to code?

While knowing how to code can enhance your understanding of design tokens, it's not a requirement for this course. The guide is designed to be accessible for both designers and developers, with practical examples that don't require coding expertise.
